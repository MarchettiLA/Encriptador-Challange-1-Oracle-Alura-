Sobre el proyecto

Este proyecto fue llevado a cabo en el marco del primer challange del curso de Alura y Oracle.

En este reto se buscaba crear una pagina web que nos permita encriptar o desencriptar un texto. Para realizar esta propuesta se utilizó Javascript, HTML5, CSS3.

Ademas se utilizaron recursos como PushbarJS y distintos tipos de tipografias de Google Fonts. A su vez el diseño tanto de la web como del logotipo y demas elementos fue tambien realizado por mi.
